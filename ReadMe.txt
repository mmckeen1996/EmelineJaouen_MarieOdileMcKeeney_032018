Nom du projet : Projet WebDesign - création d'un site affichant les évènements à Paris (débutant de 2018 à 2020)

Auteurs : Emeline Jaouen et Marie-Odile McKeeney

Etat du projet : fini le 23/03/2018

Description :
	Le site que vous pouvez désormais utiliser a été créé dans le but de pouvoir simplement consulter les évènements se produisant à Paris de 2018 à 2020. 
	La visualisation peut se faire de plusieurs manières :
		-via la carte car tous les évènements sont géolocalisés
		-via la liste recessant tous les évènements


Utilisation :

	La page est formée d’une barre de navigation grâce à laquelle l'utilisateur peut naviguer entre les deux visualisations.

	Carte :
		Lors de l'utilisation du site, la première page s'affichant présente la carte avec tous les évènements géolocalisés, l'utilisateur peut zoomer sur celle-ci en utilisant sa molette et en pointant sur la zone qui l'intéresse. Il a donc accès aux événements, représentés par des points. L'utilisateur peut cliquer sur l'un des points afin d'obtenir quelques informations sur l'évênement.
	
	Liste : 
		Lorsque l'utilisateur arrive sur la page présentant la liste, il obtient une liste des 20 premiers évènements que la liste contient. Il peut naviguer grâce à la pagination située en bas de la liste et a un indicateur de la page sur laquelle il se trouve.
		Entre la barre de navigation et la liste, l'utilisateur a accès à une barre de recherche et il peut également classer la liste de différentes manières :
			- "Du plus récent au plus ancien",
			- "Du plus ancien au plus récent",
			- Visualiser les événements commençant aujourd'hui
			- Visualiser les évênements terminant aujourd'hui
			- Visualiser les évênements dans l'ordre alphabétique
			- Visualiser les évênements dans l'ordre alphabétique inverse


Sources :
	- données : open Data (https://opendata.paris.fr/explore/dataset/evenements-a-paris/export/?disjunctive.tags&disjunctive.placename&disjunctive.city)
	- carte : mambo
	- utilisation de Bootstrap
	- utilisation d'angular.js
	- utilisation de JQuery
